    Notes

    Token-Based Authentication

  • É um protocolo usado para identificar um usuário.

  • Usado pela segurança e praticidade, onde o usuário não precisa redigitar credenciais.

  • Quando quiser acessar um conteúdo que necessita de autorização.

  • O usuário manda suas credenciais e recebe um Acess Token de volta.

    • O usuário, agora, ira usar o token para acessar as devidas páginas.
